# Sistema de Streaming Musical

Projeto avaliativo para estudo de Spring Boot e integração entre front-end e back-end.

**Objetivo:** Gerenciamento do catálogo de músicas  
**Status:** em desenvolvimento

**Problema:** Uma plataforma de streaming musical precisa de um sistema para gerenciar seu catálogo de músicas disponíveis para os usuários

O sistema deve permitir:
  - músicas com informações individuais como nome, artista, gênero, duração e ano de lançamento
  - token de streaming utilizado para reproduzir o conteúdo na plataforma 
  - operações padrão para o funcionamento básico de um serviço digital de música através de um CRUD
  - API integrada ao banco de dados
  - interação dos usuários internos através de um front-end simples
